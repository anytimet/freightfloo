Create a .env.local file in the root directory with the following content:

NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-secret-key-here-replace-with-random-string
DATABASE_URL="file:./dev.db"

# Email Service (Resend)
RESEND_API_KEY=your-resend-api-key-here

# Stripe Payment System
STRIPE_SECRET_KEY=sk_test_your-stripe-secret-key-here
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_your-stripe-publishable-key-here
STRIPE_WEBHOOK_SECRET=whsec_your-webhook-secret-here

# Google Maps Integration
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your-google-maps-api-key-here

Replace the placeholder values with your actual API keys:
- Replace "your-secret-key-here-replace-with-random-string" with a random string for security
- Get Resend API key from resend.com
- Get Stripe keys from stripe.com dashboard
- Get Google Maps API key from console.cloud.google.com
